Moverio Cardboard

Easily develop VR apps for Moverio BT 200 using Google Cardboard

Download this project as a .zip file Download this project as a tar.gz file

Welcome to Moverio Cardboard.

Now you can easily develop Virtual Reality apps for Epson Moverio BT 200. This project is a fork of Google Cardboard (0.4.10) optimized and preconfigured for Moverio. The SDK allows you to easily adapt your Unity 3D app or build a new VR experience from scratch.

Features

Moverio Cardboard leverages features found in Google Cardboard 0.4.10 such as:

  • e templates for you to use. Go ahead and click 'Continue to layouts' to browse through them. You can easily go back to edit your page before publishing. After publishing your page, you can revisit the page generator and switch to another theme. Your Page content will be preserved.

Cardboard Settings Preconfigured for Moverio

Includes Unity3D Native Android Plugin for 2D/3D mode

Native plugin handles automatic switching of 2D/3D mode on Moverio.

Or call it manually: MoverioCardboard.SDK.SetDisplay3D (true|false);
